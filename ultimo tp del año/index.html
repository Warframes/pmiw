Temática

Basicamente este juego trata sobre la película de "Angry Birds"

Proceso

Primero tuvimos que hacer una aventura gráfica respetando un diagrama de flujo para hacer dicha historia y todo esto usando por primera vez "p5.js".

Después empezamos armando el juego sobre esa historia para completar con otra forma de programación llamado POO (Programación Orientada a Objetos)
